### [Jekyll](https://jekyllrb.com/) ist ein Tool um Statische Webseiten zu erstellen....
Ideal für Blog's und auch für Webseiten die kaum dynamische Inhalte wie zum Beispiel bei Foren  
notwendig (User Verwaltung, User schreiben eigene Inhalte) beinhalten.

Nichtsdestotrotz koennen recht einfach, auf Javascript basierende, Lösungen für z.B Bildergalerien oder Schachdiagramme genutzt werden.
Andere Funktionen wie z.B Kommentare oder auch ein Kontaktformular können über Drittanbieter implementiert werden!  

Der grosse Vorteil ist gegnüber der großen [ContentManaging Systemen](https://de.wikipedia.org/wiki/Content-Management-System) wie Joomla, Wordpress,Typo3 etc -
+ **kein ständigen Sicherheits updates nötig!**  statische Seiten bringen keinen Server aus der Ruhe
+ **kein Webserver** muss zum testen Lokal installiert werden - jekyll bringt seinen eigenen mit
+ **Schreiben von Artikeln geht sehr schnell**
+ **Günstiger Webspace** ohne Datenbank und Scriptsprachen verwendbar. Über Github kostenloses Hosting möglich
und auch onhne Lokale Jekyll installation nutzbar
+ **Open Source** und frei nutzbar unter der **[MIT Lizenz](https://de.wikipedia.org/wiki/MIT-Lizenz)**
